# Less_test_stm32f103_cmsis
Development and testing of software code on stm32f103xx microcontroller using CMSIS(Розробка та тестування програмного коду на мікроконтролері stm32f103xx з використанням CMSIS).
В даному репозиторії використовується тестова плата Blue Pill на базі мікроконтролера STM32F103C8T6.
Для розробки і тестування коду використовується стандартне програмне середовище STMCubeIDE(https://www.st.com/en/development-tools/stm32cubeide.html),від компанії STMicroelectronics(https://www.st.com/content/st_com/en.html).
Для прошивки використовується утиліта ST-LINK Utility(https://www.st.com/en/development-tools/stsw-link009.html).
У прикладах(коду) тестується переферія даного мікроконтролера.
